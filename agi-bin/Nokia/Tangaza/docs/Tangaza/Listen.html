NAME

Nokia::Tangaza::Listen - Module for listening to updates

DESCRIPTION

This module allows the user to playback new and existing messages sent to a particular group.

METHODS

listen_main_menu

Plays the listening menu options for the user to determine what actions to take.

Once users select the group/vikundi whose messages they want to listen to, they have the option to listen to new, old or flagged messages. They can also flag messages for quick retrieval in future.

listen_new_menu

This only plays new messages to the user.

walk_messages_menu

Goes through all the messages the user would like to listen to based on supplied arguments (below).

Args:

$msg_type: '1' - new messages; '2' - flagged messages; 'undefined' - plays all

$channels: the group whose messages are to be played
