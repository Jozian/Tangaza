NAME

Nokia::Tangaza::Entry - Entry point into the system

DESCRIPTION

This is the entry module into Tangaza

METHODS

entry

Once the Nokia::Common::Callback::callback manpage determines the callback state and a channel with the user has been established, entry sets up the call.

It also initializes `$self` with all the necessary information by calling the Nokia::Common::Entry::entry manpage.

about_socnet_main_menu

Plays the `about page` to the user.

main_menu

This is the entry point into the system.

It plays the welcome message, lets the user know if there are any new messages to listen to and gives the user the options necessary to access all other menus (i.e. update, listen and about menu) within the system.
