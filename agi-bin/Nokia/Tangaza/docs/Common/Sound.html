NAME

Nokia::Common::Sound - Audio manager

DESCRIPTION

Manages all voice input

METHODS

init_sound

Initializes parameters necessary for processing sound files

get_language_name

Returns the name of the language based on it's language id

Args:

$language_id: The id of language whose name is being requested

msg

Uses the supplied $words parameter to loop through the file recordings in the filesystem and returns the full system filepath to be played back

Args:

$words: The name of the speech file to be played

getTTSFilename

If the file to be played-back does not exist this method generates a TTS wav file that will be played back instead

Args:

$text: The text to be converted from text to speech
