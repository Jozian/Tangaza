NAME

Nokia::Common::Stamp - Date/Time formatting and conversions.

DESCRIPTION

This module manages all time/date conversions

METHODS

year_month_day

Returns the current time into a date-time value of the form yyyymmdd

year_month_day_plus_n_days_dir

Returns a directory path of the form yyyy/mm/dd/ formed from the current date-time

Args:

$days: delta/delay adjustment to the current time

year_month_day_dir

Returns a directory path of the form yyyy/mm/dd/ formed from the current date-time

Args:

$days: delta/delay adjustment to the current time

year_month

Returns the current date-time in the form yyyy-mm

epoch2stamp

Converts the supplied epoch time to a timestamp value of the form yyyy-mm-dd hh:mm

Args:

$epoch: The epoch time to be formatted

epoch2year_mo_day

Converts the supplied epoch time to a date/time value of the form yyyy-mm-dd

Args:

$epoch: The epoch time to be formatted

file2epochdelta

Gets the statistics from the supplied file and returns the timestamp value of the date the file was last modified.

Args:

$file: The filepath to the file whose date property is to be used
