NAME

Nokia::Common::Auth - User authentication module

DESCRIPTION

This module is implements PIN-based security within the system so that any access to the system is protected from someone who might have the users phone.

METHODS

auth_pin

Authenticates the user's to the system.

If the user has a pin, it gives the user thee chances to match a four-digit pin, otherwise it does nothing.

set_pin

Allows the user to set a pin.
