NAME

Nokia::Common::Phone - Checks incoming phone number

DESCRIPTION

This module parses caller-ids for to try and determine the origin of the call. It also checks caller-ids for outgoing calls to make sure that the system only calls valid numbers e.g avoid calling 0800 numbers.

METHODS

is_valid_outbound_number

Checks whether the number is valid depending on the origin of the call

Args:

$number: The phone number being validated

parse_callerid

Parses the caller id to determine what country the call is originating from
